# Project Brief

## Project

- Name: nano-banana-family (Hide NB Studio)
- Type: Family-only image editing studio

## Core Goals

- Restrict access to allowed Google accounts.
- Provide guided image editing using Google Gemini.
- Deploy on Vercel with Next.js App Router.

## Non-Goals

- Public multi-tenant access.

## Current Priority

- 2025-12-12: Investigate and remediate React Server Components security advisory (CVE-2025-55182 et al.) for this repo.
